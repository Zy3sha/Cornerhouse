🌐 How to Update Your Website

After making changes to Sessions, Contact, About or FAQs:

1. Click the ⬇️ Download Updated Website button in any edit tab.

2. Go to your GitHub repository at github.com

3. Find index.html in the file list and click it.

4. Click the pencil ✏️ edit icon (top right of the file view).

5. Select all the text and delete it.

6. Open the downloaded file in Notepad, select all (Ctrl+A), copy (Ctrl+C), paste into GitHub.

7. Click Commit changes — your site updates in ~1 minute!


⚙️ Setting up the Booking Confirmed page (one-time setup):

1. In your SumUp dashboard, go to each Payment Link.

2. Edit the link and find "Redirect URL" or "Success URL".

3. Set it to: https://YOUR-GITHUB-USERNAME.github.io/thankyou.html

4. After paying, SumUp will send customers straight to their booking confirmation! ✅
